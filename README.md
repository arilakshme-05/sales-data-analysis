# Sales Data Analysis

## Project Overview
This project demonstrates basic *data analysis* and *visualization* using Python.  
We use *Pandas* to manipulate and analyze sales data from a CSV file and *Matplotlib* to visualize total sales by product and daily sales trends.

## Features
- Load and inspect CSV data
- Handle missing values
- Calculate total sales per transaction
- Group sales by product and date
- Generate bar and line charts for visualization
